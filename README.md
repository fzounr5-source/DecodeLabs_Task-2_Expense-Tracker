# Expense Tracker

A simple command-line Expense Tracker built in Python as part of the **DecodeLabs Internship (Industrial Training Kit) — Project 2**.

## About

This project helps track daily expenses by allowing the user to add, view, and calculate the total of all entered expenses. It focuses on core programming concepts like accumulators, loops, and basic data handling — the foundation of any backend financial logic.

## Features

- Add new expenses with name and amount
- View all recorded expenses
- Calculate and display total expenses
- Exit the program anytime

## How It Works

The program uses a simple menu-driven approach. Each expense is stored in a list, and the total is calculated using an accumulator pattern (`total += amount`). This keeps the running total updated in real time as new expenses are added.
